# Tic-tac-toe

Tic-tac-toe game on Laravel

## About the game

### Installation Instructions

Run
```bash
npm install

OR if on Homestead

npm install --no-bin-links
```

Then run
```bash
composer install
php artisan key:generate
php artisan migrate:reset
php artisan migrate
```

When finished, run the application with:
```bash
npm run dev

OR

npm run watch
```
